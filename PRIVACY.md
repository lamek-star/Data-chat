# DataChat Privacy Notice

Last updated: July 18, 2026

This repository is a standalone demonstration. Account, contact, message, and transaction information is currently stored in the user's browser. It is not transmitted to a DataChat server. Clearing browser storage removes this data.

The application uses the camera or photo picker only when the user chooses to scan a transaction QR image. The selected image is processed on-device and is not uploaded by this version.

Before commercial release, the operator must publish a jurisdiction-appropriate privacy policy identifying the legal entity, contact address, processors, retention schedule, deletion workflow, lawful bases, cross-border transfer mechanism, and user-rights process.
